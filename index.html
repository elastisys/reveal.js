Code sample

It is advisable to do work on a branch. 

git checkout -b mybranch

To prevent a flood of small commits on master, it is advisable to squash branch commits into one. For example via: 


git checkout master && git pull origin
master git checkout mybranch && git rebase -i master
# .. follow instructions
# then merge with master
git checkout master && git merge mybranch
